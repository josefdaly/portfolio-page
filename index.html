JQuery Chess

JQuery Chess is a JavaScript Chess game that utilizes JQuery to create and manipulate DOM elements to graphically represent the game. This implementation, like my implementation of Asteroids, was an exercise in Object Oriented JavaScript following the prototype-oriented model and makes extensive use of prototypal inheritance.

At present, JQuery Chess only supports two player games. Stay tuned for a computer AI. I should implement it in the next few days.

ReadMe

ReadMe is a full-stack web app with a Backbone.js front-end that consumes a RESTful Rails API serving models in JSON. ReadMe's primary purpose is to allow users to upload, read, and rate EPub files.

My favorite part of ReadMe is the e-reader, which allows a user to read any uploaded epub (preferably one that they wrote themselves). The reader was built using the JavaScript library EPub.js by Future Press.

As ReadMe is mostly a demonstration application, the library, while intended for independent publishing, is seeded with great works by writers who are long dead (along with their proprietary ownership over their works.) Feel free to log in and read, rate, and upload your own books. A guest account feature is provided on the log in page for your convenience.

Asteroids

Asteroids is a JavaScript game inspired by the arcade classic of the same name. The game is rendered on an HTML5 canvas and uses JQuery to update the scoreboard. This implementation was an exercise in Object Oriented JavaScript following the prototype-oriented model and makes extensive use of prototypal inheritance.

Unlike the original game, this Asteroids uses the modern WASD/Mouse control scheme.

Towers of Hanoi

This implementation of Towers of Hanoi was one of my first exercises in JQuery. The goal of the game is to move all of the discs from one pile to another. The only restriction is that you cannot place a larger disc atop a smaller disc.

Ruby Chess

Ruby Chess is a simple CLI chess game written in Ruby emphasizing Object Oriented best practices. Ruby Chess supports human vs AI, AI vs AI, and human vs human games.

Ruby Chess is available as a Ruby Gem with the command 'gem install ruby_chess'.
